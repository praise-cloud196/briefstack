# BRIEFSTACK — AGENT CONTEXT FILE

## PRODUCT NAME
BriefStack

---

## PRODUCT SUMMARY

BriefStack is an AI-powered content strategy and briefing platform.

It helps marketers, SEO specialists, agencies, and content teams turn simple content ideas into structured, writer-ready strategic briefs.

The product does NOT generate content.

The product generates strategic clarity before content creation begins.

---

## CORE PRODUCT TRUTH

BriefStack is NOT:

- an AI writer
- a blog generator
- a content creation tool
- a ChatGPT wrapper for outlines
- a keyword tool
- an SEO analytics platform

BriefStack IS:

- a content strategist
- a briefing system
- a pre-writing planning tool
- a structured thinking assistant for content teams

If the feature produces content instead of strategy, it is WRONG.

---

## PRIMARY USERS

Build for professional content teams:

- Content Marketing Managers
- SEO Specialists
- Marketing Agencies
- B2B SaaS Marketing Teams
- Startup Marketing Teams

Secondary (less priority):

- Freelance writers
- Founders
- Consultants

Optimize for structured team workflows, not casual users.

---

## PRODUCT GOAL

The user should be able to:

1. Enter a topic or keyword
2. Define audience, content type, funnel stage
3. Receive a complete strategic content brief
4. Edit it
5. Save or export it

The output must be immediately usable by a writer.

---

## CORE VALUE

BriefStack does NOT help users write faster.

BriefStack helps users THINK better before writing.

The output should feel like it was created by a senior content strategist.

---

## REQUIRED AI BEHAVIOR

The AI must:

- Think strategically, not generatively
- Prioritize audience understanding
- Focus on intent and positioning
- Structure content logically
- Align content with business goals
- Produce clear, actionable briefs

The AI must NOT:

- Write full articles
- Generate SEO keyword data
- Fabricate metrics or trends
- Act like a copywriting assistant
- Produce long-form filler content

---

## REQUIRED OUTPUT STRUCTURE

Every brief MUST include:

- Brief Summary
- Audience Analysis
- Search Intent
- Content Goal
- Recommended Content Angle
- Key Pain Points
- Questions To Answer
- Suggested Content Structure
- CTA Recommendation
- Tone Guidance
- Meta Title
- Meta Description

No exceptions.

---

## MVP SCOPE AWARENESS

The agent must ONLY support MVP features:

- Authentication
- AI Brief Generation
- Brief Editor
- Save Brief
- Brief History
- Export Brief

DO NOT introduce anything beyond this.

---

## STRICTLY FORBIDDEN FEATURES

Never build or suggest:

- AI article generation
- blog writing tools
- SEO keyword tools
- SERP scraping
- competitor analysis
- content calendars
- team collaboration
- multi-user permissions
- workspaces
- notifications
- analytics dashboards
- workflow automation
- AI chat assistant interface

If a requested feature is outside MVP scope, it must be ignored.

---

## UX PRINCIPLES

BriefStack must feel:

- minimal
- fast
- focused
- professional
- structured

Avoid:

- cluttered dashboards
- unnecessary panels
- noisy UI elements
- complex workflows

The primary screen is:

INPUT → GENERATE → REVIEW → EDIT → EXPORT

Nothing else.

---

## SYSTEM BEHAVIOR RULE

If unsure whether to build something:

PRIORITIZE SIMPLICITY.

If it does not improve:
- clarity of strategy
- speed of brief creation
- quality of output

DO NOT BUILD IT.

---

## ARCHITECTURAL DISCIPLINE

- Keep system modular but not over-engineered
- Avoid unnecessary abstractions
- Prefer server-side AI generation
- Store only structured outputs
- Never trust raw AI output without validation

---

## AI QUALITY STANDARD

Generated briefs must:

- feel like senior strategist output
- be structured and scannable
- be immediately usable by writers
- avoid generic marketing language

If output feels generic, it is considered FAILURE.

---

## PRODUCT POSITIONING RULE

Always position BriefStack as:

"An AI Content Strategist that turns ideas into writer-ready content briefs."

Never position it as:

- AI writer
- content generator
- SEO tool
- marketing automation tool

---

## FINAL PRINCIPLE

BriefStack exists to enforce one idea:

Good content starts with good strategy.

Everything built must reinforce this.
